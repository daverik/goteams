1. Starta Wamp/Mamp.
2. Öppna din web browser och skriv in: localhost/phpmyadmin.
3. Logga in (lösenordsfältet ska troligtvis lämnas tomt).
4. Byt lösenord genom att klicka på länken "Byt lösenord" som finns på första sidan. Ändra lösenord till "root".
5. Klicka sedan på fliken "Importera".
6. Klicka på "Välj fil" och leta upp "lagvarvet.sql" i mappen "Database" i goteams. Klicka "Kör".
7. Done!
